# 胧编辑·md助手 需求文档 (PRD) - 最终顶层设计版

## 1. 产品概述
针对 Windows 平台设计的极简美学 Markdown 编辑器。核心定位：**双模式切换（单文件/知识库）** + **苹果级视觉体验** + **镜像物理同步**。

## 2. 核心功能需求

### 2.1 双模式与单实例架构 (Single Instance)
- **单实例运行**：Windows 下双击多个文件，始终在已有的唯一窗口中通过多标签页 (Tabs) 开启，避免多开乱象。
- **镜像物理同步**：软件库内的目录结构与 Windows 资源管理器的物理文件夹 1:1 镜像同步，确保数据透明、易于备份。

### 2.2 极致写作与交互
- **专注/打字机模式**：F11 进入禅定模式，配合打字机居中显示。
- **悬浮格式工具栏**：基于选中内容的上下文菜单，药丸型毛玻璃设计。
- **智能粘贴**：链接抓取网页标题、Excel 转 Markdown 表格、图片自动入库。
- **浮动预览 (Hover Preview)**：鼠标悬停在 `[[双向链接]]` 上时，弹出一个毛玻璃小窗显示目标文件的内容预览，无需跳转。

### 2.3 知识库管理
- **命令面板 (Ctrl+P)**：全局搜索、命令执行（如 `>导出`）。
- **库内全文检索**：高性能本地索引，支持关键词实时高亮。
- **可视化 YAML 编辑器**：侧边栏卡片式管理 Frontmatter。

### 2.4 数据安全与高级工具
- **版本时间轴 (Version Timeline)**：基于影子副本机制，记录文件的修改历史，支持一键回滚到之前的某个保存点。
- **多格式导出**：PDF、HTML、Word (.docx)、长图 (Image)。
- **Windows 资源管理器集成**：右键菜单添加“导入到 MarkDownReader 库”选项。

### 2.5 托盘快速收集 (Quick Capture)
- 系统托盘图标支持右键“快速笔记”，弹出一个极简的小窗，记录后自动存入库中的 `Inbox` 文件夹。
