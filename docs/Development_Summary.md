# 胧编辑 · md助手 历史记录 (2026-03-04 - 大纲与同步重构版)

## 1. 当前项目状态
项目已成功实现高度稳定的右侧大纲同步系统与可靠的影子副本存档逻辑。通过“推倒重构”方案，彻底解决了 Vditor 外部大纲渲染空白及多文档并发保存冲突的疑难杂症。

### 已实现核心功能：
- **实时大纲系统 (Manual Extraction)**：
    - **原理**：放弃 Vditor 官方 `outlineRender`，改用原生 DOM 扫描 + `MutationObserver` 监听。
    - **亮点**：自动为缺失 ID 的标题注入锚点，支持 H1-H6 阶梯缩进，毫秒级响应输入变化。
- **影子副本 (Shadow Copy) 3.0**：
    - **路径校验锁**：引入 `lastLoadedPath` 原子锁，确保自动保存的内容与当前激活的文件 100% 匹配。
    - **内容归一化**：比对时自动忽略 Windows/Unix 换行符差异，杜绝冗余气泡。
- **知识库智能关联**：修复了启动时因配置加载延迟导致的“未关联”显示 Bug。

## 4. 主题同步与系统深度集成 (2026-03-04)
修复了多主题切换中的背景同步失效问题，并完成了快速捕获小窗与系统右键菜单的集成。

### 已实现核心功能：
- **UI 动效系统全量升级 (Premium UI Experience)**：
    - **深度图层转场 (Depth Switch)**：重构全局路由切换动效，实现新旧页面并行交替。入场页面采用 `scale(0.96) + 15px` 垂直滑入，离场页面轻微放大渐隐，营造出极具空间感的层级体验。
    - **极速响应 (0.15s - 0.3s)**：移除串行等待模式，配合 `cubic-bezier(0.16, 1, 0.3, 1)` 曲线，确保每一次点击都有干脆且优雅的反馈。
    - **阶梯式加载 (Staggered Load)**：在欢迎页、设置板块和影子副本气泡中应用 staggered 动画，提升页面展开的仪式感。
    - **交互微交互 (Micro-interactions)**：为侧边栏折叠按钮、标签页位移、命令面板弹出及悬浮预览注入灵动特效。
    - **性能与稳定性优化**：针对文件树展开时的闪烁问题，将节点级动画升级为容器级一次性动画，确保大规模库操作的稳健性。

## 5. 系统集成与交互闭环优化 (2026-03-04)
完成了从“单窗口工具”到“常驻系统应用”的体验跨越，解决了托盘交互、退出拦截及多窗口联动的核心痛点。

### 已实现核心功能：
- **常驻托盘体系**：
    - **交互纯净化**：在 Rust 后端限制 `TrayIconEvent::Click`仅响应左键点击，彻底解决了右键菜单闪烁消失的顽疾。
    - **悬停信息补全**：为托盘图标添加了 `.tooltip()` 支持。
- **自定义退出确认机制**：
    - **无侵入式 Modal**：通过手写极简 Vue 遮罩层实现退出确认，避免了引入全局 Provider 导致的样式污染与布局偏移。
    - **隐藏即保活**：后端拦截 `CloseRequested` 并默认执行 `window.hide()`，确保应用在后台持续运行。
- **快速笔记 (Inbox) 深度联动**：
    - **跨窗口自动刷新**：基于 Tauri `emit/listen` 事件机制，实现快速笔记保存后自动触发主界面库刷新。
    - **智能时间戳命名**：引入 `快速笔记_YYYYMMDD_HHMMSS` 命名规则，解决重名冲突并自带时间属性。
    - **响应式布局适配**：利用 `flex-shrink: 0` 和 `min-height: 0` 解决了小窗口下操作按钮被挤出的布局 Bug。

### 踩坑点总结 (The Hard Way)
| 坑点场景 | 根本原因 | 最终处理方式 |
| :--- | :--- | :--- |
| **托盘菜单闪烁** | `TrayIconEvent::Click` 默认包含所有按键，右键点击也触发了窗口聚焦逻辑。 | **按键过滤**：在 Rust 端显式判定 `MouseButton::Left`。 |
| **窗口无法隐藏/销毁** | Tauri 2.0 权限管控极严，未在 `capabilities` 中授权 `allow-hide/destroy`。 | **权限补全**：在 `default.json` 中扩展权限清单并授权给所有窗口标签。 |
| **Provider 外部调用报错** | `useDialog` 等 Hook 无法在定义 Provider 的组件 setup 中直接使用。 | **轻量化/Discrete API**：改用手写 Modal 或 Naive UI 的独立 API 避开层级限制。 |
| **快速笔记找不到路径** | 库路径字段在重构中已更名，且新窗口未在 `onMounted` 重新同步配置。 | **Store 预加载**：在新窗口初始化时强制执行一次 `store.loadConfig()`。 |
