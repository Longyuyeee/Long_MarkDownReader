# 胧编辑·md助手 开发里程碑规划 (Milestones)

## 📍 Milestone 1: 工程基建与 UI 框架搭建 (✅ 已完成)
- [x] 初始化 Tauri 2.0 + Vue 3 + TypeScript 跨平台项目脚手架。
- [x] 配置 Vite、Tailwind CSS (或原生 CSS) 及 Naive UI 组件库。
- [x] 实现 Windows 无边框窗口 (Borderless) 及其控制按钮（最小化、最大化、关闭）。
- [x] 接入 Windows 11 Mica (毛玻璃) 材质，奠定整体“果味”视觉基调。
- [x] 设置 Vue Router，预留 `/temp` (临时模式) 和 `/library` (库模式) 路由。

## 📍 Milestone 2: 核心编辑器引擎与文件 I/O (✅ 已完成)
- [x] 集成 Vditor (或选择的 Markdown 引擎)，实现所见即所得、双栏、打字机模式。
- [x] 实现 Rust 后端的本地文件读取 (Open) 与覆盖保存 (Save) 功能。
- [x] 实现通过命令行参数启动（双击 `.md` 文件直接打开并载入内容）。
- [x] 完成**临时编辑模式 (Temp Mode)** 的基础界面交互。

## 📍 Milestone 3: 软件库模式与文件系统同步 (✅ 已完成)
- [x] 构建**软件库模式 (Library Mode)** 的经典三栏布局（侧边栏、编辑器、属性栏）。
- [x] 开发左侧文件树组件，实现与本地操作系统的物理目录 1:1 镜像同步。
- [x] 实现“临时转库”的物理导入逻辑（复制模式，提示并转移到指定文件夹）。
- [x] 实现多标签页 (Tabs) 的状态管理 (Pinia) 与切换逻辑。
- [x] **新增**：实现物理“新建笔记”功能，自动解决重名冲突。

## 📍 Milestone 4: 资产管理与智能粘贴体验 (✅ 已完成)
- [x] 实现 Markdown 专用图片文件夹 (`.assets`) 策略。
- [x] 导入库时，Rust 后端解析原文件图片路径、物理搬迁图片并重写源码。
- [x] 剪贴板事件劫持 (Smart Paste)：网页链接自动转 Markdown 格式（后端抓取 Title），Excel 转表格。
- [x] 粘贴/拖拽图片时，自动存入 `.assets` 目录并生成相对路径。

## 📍 Milestone 5: 进阶体验与知识管理 (✅ 已完成)
- [x] 开发命令面板 (Command Palette，`Ctrl+P`) 与局部/全局全文检索（当前支持文件名搜索）。
- [x] 实现双链解析 (`[[ ]]`)，以及浮动内容预览 (Hover Preview)。
- [x] 专注模式 (Zen Mode) 与全屏动效。
- [x] 影子副本 (Shadow Copy) 自动保存机制。
- [x] **新增**：实现设置弹窗，支持库路径修改与主题持久化切换。

## 📍 Milestone 6: 系统深度集成与发布打包 (✅ 已完成)
- [x] 托盘图标 (System Tray) 支持一键呼出与退出。
- [x] 快速捕获 (Quick Capture) 小窗开发 (已实现基础框架与 Inbox 存入逻辑)。
- [x] Windows 资源管理器右键菜单深度集成 (已实现注册表脚本注入与默认打开方式关联)。
- [x] 单实例运行 (Single Instance) 优化，双击新文件打开为新标签页。
- [x] 文档导出功能集成 (已支持 HTML)。
- [x] 配置持久化存储 (已实现核心框架，正在优化 Vditor 模式与代码主题的重启恢复机制)。
- [ ] 整体内存与性能测试，生成最终的 Windows 安装包 (.msi 或 .exe)。
